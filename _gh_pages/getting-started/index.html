JokerJS

JokerJS foi desenvolvido para facilitar a criação de uma aplicação RIA voltada para a área de sistema. Ele busca a facilidade de gerar códigos e padronização da aplicação.

JokerJS nasceu para aplicações RoR ( Ruby on Rails ) o qual possibilitou total integração a gama de recursos que esta maravilho framework disponibiliza.

JokerJS nasceu para aplicações RoR ( Ruby on Rails ) o qual possibilitou total integração a gama de recursos que esta maravilho framework disponibiliza.

Dependências

O JokerJS é um framework que utiliza varias dependencias para existir, desde o seu funcionamento ao sistema de teste automatizado. Confira abaixo a listagem completa de dependencias do JokerJS:

Ruby on Rails
Onde a aplicacao deve rodar.
CoffeeScript
Linguagem utilizada nas bibliotecas do Joker para compilar o javascript.
SCSS
Utilizado para servir o CSS, e possibilitando a mudança rápido das cores.
jQuery ou Zepto.js
Biblioteca que serve recursos ao Joker efetuar as manipulações que necessita.

Instalando no Rails

É preciso adicionar no arquivo: Gemfile

gem 'joker-rails'

Após adiciona ao Gemfile deve ser executado para a instalação da GEM, o seguinte comando:

$ bundle install
