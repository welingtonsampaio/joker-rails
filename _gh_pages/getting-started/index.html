JokerJS

JokerJS foi desenvolvido para facilitar a criação de uma aplicação RIA voltada para a área de sistema. Ele busca a facilidade de gerar códigos e padronização da aplicação.

JokerJS nasceu para aplicações RoR ( Ruby on Rails ) o qual possibilitou total integração a gama de recursos para este maravilhoso framework disponibiliza.

Por que Joker

Joker veio através de uma decepção. Nossa, mas por que uma decepção? Chatiados com a má documentação do BatmanJS, a equipe resolveu criar o Joker.

Instalação

Dependências

O JokerJS é um framework que utiliza varias dependências para existir, desde o seu funcionamento ao sistema de teste automatizado. Confira abaixo a listagem completa de dependências do JokerJS:

Ruby on Rails
Onde a aplicação deve rodar
CoffeeScript
Linguagem utilizada nas bibliotecas do Joker para compilar o javascript.
SCSS
Utilizado para servir o CSS, possibilitando a rápida mudança das cores.
jQuery ou Zepto.js
Biblioteca que serve recursos ao Joker para efetuar as manipulações que necessita.

Instalando no Rails

No arquivo Gemfile do Rails, é preciso adicionar o arquivo do JokerJs:

$  gem 'joker-rails'

Após adicionar ao Gemfile, deve ser executado o seguinte comando para a instalação da Gem: 

$  bundle install

Pronto, a instalação já foi concluída.

Criando uma aplicação JokerJS

Com o comando rails new podemos instalar os pacotes necessários para a criação de uma aplicação JokerJS 

$ rails new Carro

(No nosso exemplo, estamos utilizando Carro)

Logo após a criação, deve ser executado o seguinte código: 

$ bundle install

O próximo passo é gerar os arquivos do JokerJS 

$ rails generate joker:install

Aí, você cria sua aplicação contendo o que você deseja: 

$ rails generate joker:scaffold Carro cor:string ano:integer marca:string placa:string

(Neste caso, foi criado um objeto Carro, com os atributos cor, ano, marca e placa)

Para criar o banco de dados da sua aplicação, é necessário executar o seguinte código: 

$ rake db:migrate

Depois, faça o bundle instalar isso: 

$ bundle install

Aplicação criada, agora é só utilizá-la.
